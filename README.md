# E-Commerce Sales and Transaction Analytics

Python-based data analytics project analyzing e-commerce sales trends, transaction patterns, regional performance, and business insights using Google Colab.

## Features
- Sales trend analysis
- Regional performance insights
- Product category analysis
- Monthly revenue trends
- Payment method analysis
- Transaction pattern visualization
- Heatmap and pivot table insights

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## Dataset
Online sales transaction dataset containing:
- Order ID
- Date
- Product category
- Quantity sold
- Revenue
- Region
- Payment method

## Key Insights
- Identified top-performing product categories
- Analyzed seasonal sales peaks
- Compared regional sales performance
- Derived payment behavior insights
